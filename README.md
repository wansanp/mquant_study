# M Quant
## Environment 
   - Anaconda 3.7
   - Pycharm C.E
   
## 디렉토리 설명
   - book : 책 예제
   - data : 입력 데이타 등 미리 주어져서 왠만하면 변하지 않는 데이터들
   - raw_data_access : fnguide, krx, naver 등에서 raw 데이터를 추출
   - raw_data_process : 위의 데이터를 바탕으로 계산, 논리적인 비교 등 포함
   - request : 요청에 의해 custom 으로 구현한 것들
   - result : 실험에 의해 나온 결과들
